# Cake Delight

## Description
A website about cakes with recipes, gallery, tips, and contact page. Built with HTML, CSS, and JS. Some pages also demonstrate React components.

## Features
- Homepage with hero section and featured cakes.
- Navigation bar linking all pages.
- Gallery and About pages.
- Contact page with form and subscribe form.
- Responsive design using Bootstrap.
- Footer with copyright.

## Pages
- `index.html` – Home
- `about.html` – About
- `gallery.html` – Gallery
- `recipes.html` – Recipes
- `contact.html` – Contact
- `subscribe.html` – Subscribe form

## How to run
1. Open `index.html` in your browser.
2. Make sure CSS files are in the `css` folder and JS files in `js` folder.
3. Use Live Server in VS Code for better experience.
